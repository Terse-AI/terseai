# Security Policy

Terse runs **100% on-device** — your prompts and code never leave your machine. We take the security of the Terse desktop app, the Chrome and VS Code extensions, and the SDK in this repo seriously.

## Reporting a vulnerability

Please **do not** open a public issue for security reports. Instead:

- Email **security@terseai.org**, or
- Open a private [GitHub Security Advisory](https://github.com/Terse-AI/terseai/security/advisories/new)

We aim to acknowledge reports within **3 business days** and to provide a fix or mitigation timeline after triage.

## Scope

- Terse for macOS & Windows (the desktop app)
- The Terse Chrome extension and VS Code extension
- The Terse SDK in this repository

Thank you for helping keep Terse and its users safe.
